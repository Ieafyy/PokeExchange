Sobre o projeto

Olá! Seja bem vindo ao poketrade! Um projeto desenvolvido por mim para treinar os meus conhecimentos trabalhando com APIs. O sistema foi escrito em Python e seu backend no framework Flask. As APIs usadas foram: PokeApi e Awesome Api. A ideia do projeto é pegar os 3 primeiros digitos (diferentes de 0) da taxa de câmbio das principais moedas com o Real e apartir desse número gerado, obter o Pokemon com ID correspondente. Se gostou, dê uma olhada em meu Github!
