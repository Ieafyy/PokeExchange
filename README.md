# PokeExchange

![image](https://user-images.githubusercontent.com/70926962/172280254-274d2b93-acdf-4bf4-834c-5d069213dff1.png)

Projeto escrito em Python e que comunica com as APIs <a href="https://pokeapi.co/">PokeAPI</a> e <a href="https://docs.awesomeapi.com.br/api-de-moedas">Awesome API</a> para obter a taxa de cambio de divesas moedas para real (BRL) e mostrar como resposta o Pokemon referente aos 3 primeiros dígitos da taxa de cambio. 

Nesse projeto foi usado a lib requests para puxar dados das APIs citadas e os resultados são apresentados numa interface web: o Backend escrito em Python (usando Flask) e o frontend escrito em HTML/CSS (Tailwind Css).

Além disso, este projeto foi postado na web usando Git + Heroku e pode ser acessado em: https://pokeexchange.herokuapp.com/
